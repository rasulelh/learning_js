# Chess Board Generator

Generates a two-dimensional chess board with alternating cells.

---

## ðŸ“Œ Description

The function `generateChessBoard` creates a square chess board
of size `x * x` represented as a two-dimensional array.

White cells are marked with `"."` and black cells with `"#"`.
The color of each cell is determined by the sum of its row and column indexes.
